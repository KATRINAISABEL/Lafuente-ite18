# Lafuente-ite18

## ITE18 Activity 1 & 2 - Personal Website

A personal portfolio website created for ITE-18 coursework, featuring multiple pages with modern styling and responsive design.

### Project Structure

- **index.html** - Home page with personal information
- **about.html** - About page with skills list and layout exercise
- **style.css** - Comprehensive styling for the entire project
- **README.md** - Project documentation

### Features

#### Navigation
- Sticky navigation bar on all pages
- Active page indicator
- Smooth hover effects

#### About Page
- Personal introduction (3-5 sentences)
- Skills and interests list
- Responsive design

#### Layout Exercise
- 6 colored boxes arranged with flexbox
- Centered heading
- Hover effects with scale and shadow transformations
- No scrollbar overflow

#### CSS Styling
- Modern color scheme
- Typography improvements
- Flexbox layout system
- Hover effects and transitions
- Box shadows for depth
- Rounded corners for modern appearance
- Responsive design for mobile devices

### Technologies Used
- HTML5
- CSS3 (Flexbox, CSS Transitions)
- Git for version control

### How to Run
Simply open `index.html` in a web browser to view the website locally.
